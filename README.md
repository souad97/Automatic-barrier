# Automatic-barrier
realisation d'un barriere automatique 
HARDWARE : 

       - ARDUINO UNO 
       - ultrason 3
       - LCD 
       - servomotor 
       - fils de connexion 
       - 2 leds : rouge et verte 
       - cable HDMI pour l'arduino 
       - breadboard
